== Description ==

IceCream Elementor Addon adds extra widgets to your Elementor page builder.

== IceCream Widgets ==

- **Image Gallery**. Add beautiful and responsive mosaic image gallery to your page.

- **Image Hover**. Add image with hidden title and description which show details on hover.

- **Menu Image**. You able to create a hidden menu over an image. Hidden menu shows on hover user action.

- **Expanding Button**. Can create expanding button with an icon in it which fully customizable.

- **Team Member 1**. Team member card - hover social media animation.

- **Team Member 2**. Team member card - hover card details.

- **Team Member 3**. Team member card - direction effect.

- **Team Member 4**. Team member card - big card data.

- **Team Member 5**. Team member card - fluid information on card.


== Frequently Asked Questions ==

**Is this plugin require any plugin to be installed?**

Yes. Since this plugin extend widget from Elementor. Elementor is required..

**Can I use this plugin with any themes?**

Yes, you can. This plugin works with any themes as long you use Elementor and the theme compatible with Elementor.

**After the update, some styles have changed and looks weird. What can I do to fix it?**

Please try to regenerate CSS via Elementor > Tools > Regenerate CSS option on admin dashboard.

**My Elementor editor isn't loading. What should I do?**

Please make sure the WP Memory Limit on your site has set to 128 MB or greater. You can check your site status on Elementor > System Info. Please read this article for further reading
https://elementor.com/help/requirements/ Also you can disable some widgets that you don't need from Jeg Elementor Kit > Elements dashboard to reduce memory usage.
